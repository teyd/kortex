# v0.5.0

- **Smarter Process Matching**: Improved detection logic to handle case sensitivity and missing file extensions (e.g., matching "CS2" to "cs2.exe").
- **Robust Configuration**: Switched to a strict, typed schema managed by Rust for better stability.
- **Flicker-Free Startup**: Optimized window initialization to prevent the flash when starting minimized.
- **UI Enhancements**: Added an application version tag to the Settings screen.
- **Single Instance**: Prevented multiple instances of the application from running simultaneously.
